# Enterprise Multi-Agent OS Starter Kit

This starter kit contains the AI development framework, Phase 1 specs and demo dataset for the project:

Enterprise Multi-Agent OS: Automated Business Workflow Orchestration

Academic title:

Enterprise Procurement Workflow Automation using LangGraph-based Multi-Agent System

## How To Use With Codex/Harness

1. Copy the `.ai` directory into the GitHub repository root.
2. Copy `datasets` and `docs` into the repository root.
3. Ask Codex to read:
   - `.ai/project/PROJECT_CONTRACT.md`
   - `.ai/project/ARCHITECTURE.md`
   - `.ai/project/TECH_STACK.md`
   - `.ai/project/CODING_STANDARD.md`
   - `.ai/specs/SPEC-001-bootstrap-backend/spec.md`
   - `.ai/specs/SPEC-001-bootstrap-backend/tasks.md`
4. Start with TASK 001.1 only.
5. Do not ask Codex to implement all specs at once.

## First Codex Task

Implement TASK 001.1 from SPEC-001.

Codex must only create the backend skeleton, pyproject.toml, base FastAPI app and README.
